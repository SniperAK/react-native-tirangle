# React Native Triangle View

## Install 

``` 
$ npm install react-native-triangle-view
```

## Usage


```
...
import Triangle from 'react-native-triangle-view';


... render somewhere

<Triangle 
  width={10}
  height={10}
  direciton='up'
  color='red'
/>

```

## Property Types

- direction : 
  - up
  - right
  - left
  - down
  - up-right
  - up-left
  - down-right
  - down-left
- width
- height
- color
